# SSR(服务器渲染)

SSR: 服务端渲染 在服务器对vue页面进行渲染HTML文件 返回给浏览器

# SSR优点

SEO : 不同于SPA的HTML只有一个无实际内容的HTML和一个app.js，SSR生成的HTML是有内容的，这让搜索引擎能够索引到页面内容。

更快内容到达时间 传统的SPA应用是将bundle.js从服务器获取，然后在客户端解析并挂载到dom。而SSR直接将HTML字符串传递给浏览器。大大加快了首屏加载时间。
